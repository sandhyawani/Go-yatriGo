🌍 Go YatriGo - Connect. Travel. Explore Together.**

Go YatriGo is a full-stack **MERN** travel companion platform that connects travelers with like-minded people. Users can discover travel groups, create their own trips, send join requests, chat in real time, and manage their travel experiences through a secure and responsive web application.

## 🚀 Features

- 🔐 Secure JWT Authentication & Authorization
- 👤 User Registration & Profile Management
- 🌍 Create and Manage Travel Groups
- 🔎 Explore & Search Travel Groups
- 🤝 Send, Accept & Reject Join Requests
- 💬 Real-Time Chat using Socket.IO
- ❤️ Save & Manage Favourite Travel Groups
- 📱 Fully Responsive Design
- ⚡ RESTful API Architecture
- 🎨 Modern UI with Tailwind CSS & Shadcn UI
- 🔄 Redux Toolkit for State Management

### Navigate to Project

```bash
cd go-yatrigo
```

### Install Frontend Dependencies

```bash
cd client
npm install
```

### Install Backend Dependencies

```bash
cd ../server
npm install
```

### Configure Environment Variables

Create a `.env` file inside the **server** folder.

```env
PORT=8000

MONGO_URI=your_mongodb_connection_string

JWT_SECRET=your_secret_key

CLIENT_URL=http://localhost:5173

CLOUDINARY_CLOUD_NAME=your_cloud_name

CLOUDINARY_API_KEY=your_api_key

CLOUDINARY_API_SECRET=your_api_secret
```

## ▶️ Run the Project
